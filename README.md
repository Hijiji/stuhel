# stuhel
div학업을 도와주는 <strong>study helper [STUHEL]</strong> 입니다.</br>
과거 학원을 다니던 시절, 팀모임을 위한 세미나실 예약에 불편함을 경험하여 이를 해결하고자
세미나실 예약을 주 목적으로 하는 본 프로젝트를 개발했습니다.

<strong>제공기능</strong>
1. 로그인-로그아웃 기능
2. 세미나실 예약-확인-취소 기능
3. 정보공유를 위한 게시판 기능-댓글기능
4. 개인정보수정기능

<strong>사용된 스킬</strong>
1. Java
2. JavaScript
3. Oracle
4. CSS
5. HTML
6. MyBatis
